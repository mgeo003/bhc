About These Mockups

This is a comprehensive set of interactive HTML/CSS prototypes for BehavioralHealth Connect, a B2B SaaS platform connecting referring clinicians with behavioral health treatment providers.

These mockups demonstrate complete user flows for both clinicians and providers with a minimal, clean, healthcare-appropriate design. All interactions are functional using vanilla JavaScript.

Technical Details

Technologies Used

  • HTML5 semantic markup
  • Tailwind CSS (CDN)
  • Vanilla JavaScript
  • Inter Font (Google Fonts)
  • Heroicons (inline SVG)

Design Principles

  • Mobile-First: Responsive design optimized for all devices
  • Accessible: WCAG AA contrast, keyboard navigation
  • Professional: Healthcare-appropriate minimal design
  • Interactive: Functional forms, modals, and state management
  • Consistent: Unified design system across all screens
